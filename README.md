# Film Rating Website
## Giới thiệu
Website cung cấp thông tin phim và cho phép người dùng tìm kiếm, xem chi tiết và đánh giá phim.
## Demo
https://zeronocolor-netizen.github.io/film-rate/
## Chức năng
- Tìm kiếm phim
- Xem danh sách phim
- Xem chi tiết phim
- Đánh giá phim
## Công nghệ
- HTML
- CSS (Flexbox, Grid)
- JavaScript (DOM, LocalStorage)
- JSON
